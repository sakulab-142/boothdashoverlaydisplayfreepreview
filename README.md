# SAKULAB 配信素材セット 導入ガイド

この度は SAKULAB の製品をご利用いただき、誠にありがとうございます。
本製品は、ブラウザ操作パネルから配信画面をリアルタイムにカスタマイズできるHTMLベースの配信素材です。

--------------------------------------------------
■ パッケージ内容
--------------------------------------------------
・display.html       ： OBSに読み込む「表示用」ファイル
・control_lite.html  ： 無料版 操作パネル（機能制限あり）
・control_pro.html   ： 有料版 操作パネル（全機能解放）
・利用規約.txt          ： 利用規約（必ずご確認ください）

--------------------------------------------------
■ 導入手順
--------------------------------------------------
1. 【操作パネルの準備】
   Google Chrome等のブラウザで「control_lite.html」（または有料版）を開きます。
   配信中にすぐ操作できるよう、ブラウザのブックマークに登録することを推奨します。

2. 【OBS Studioへの設定】
   ① OBSのソース一覧の「＋」から「ブラウザ」を選択。
   ② 「ローカルファイル」にチェックを入れ、「参照」から「display.html」を選択。
   ③ 幅を「1920」、高さを「1080」に設定してOKを押します。
   ④ ゲーム画面やカメラ映像の上に配置してください。

3. 【カスタマイズ】
   ブラウザで開いた操作パネルから、色、フォント、テキストを変更してください。
   入力した内容は即座にOBS側の画面へ反映されます。

--------------------------------------------------
■ 無料版と有料版（PRO）の違い
--------------------------------------------------
【無料版】
・カラー選択：プリセット3色のみ
・フォント：2種類のみ
・テロップ：最大20文字まで
・16進数による詳細な色指定不可

【有料版】
・すべての制限を解除
・16進数による自由なカラー設定
・全フォント選択可能
・文字数制限なし
